# Food App

This repository contains a Vite + React + TypeScript starter for a food donation/listing application.

Getting started:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

Tech stack:

- Vite
- TypeScript
- React
- Tailwind CSS

To run tests and linters:

```sh
npm run lint
npm test
```
